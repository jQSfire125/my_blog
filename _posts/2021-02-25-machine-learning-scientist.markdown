Today I finished the Machine Learning Scientist with Python Career track in Data Camp: 23 courses, an estimated 93 hours of instruction. It took me 5 months. Done!

I still don't believe I am a data scientist yet. But this was a good sized step. I really liked how scikit-learn uses a standard way of doing things, that makes the learning a lot easier. I also enjoyed very much Data Camp's hands-on modules. They make learning so fun.

Keras and Tensorflow were very cool too. The first time I saw a nerual network described in [3Blue1Brown][3b1b] last July, I was intimidated. But after a couple modules in Data Camp, plus the course I took in Kaggle, I like NN very much. Keras makes it easy to setup, and even at the basic level of these courses, there are a lot of things one can do. I want to spend more time in Kaggle to learn more and join a couple of the competitions that use neural networks.

Back in September I signed up to a 6 week program Machine Learning Accelerator with [Elite Data Science][eds]. I will start that one tomorrow. Let's see how it is.

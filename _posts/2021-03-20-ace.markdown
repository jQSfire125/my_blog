I am very happy to report that I recently received my work permit. It is a one-year permit while the Green Card process is complete. I will transition at Strata (my current company) to more of an advisory role over by the end of May. And I will start looking for a position in the data science field by then.

As part of that exploration, I talked to a really nice guy who graduated a couple of years ago from a data science masters at Berkley. He suggested I complemented my training and start learning more about the Google cloud platform. 

I believe he really knows what he is talking about and so I signed up for the Google Certified Associate Cloud Engineer certification course at [A Cloud Guru][acg]. We'll see how it goes.

On the data science front, it turns out the [Elite Data Science][eds] course is just **excellent**, and is just what I needed to cement the knowledge and put it all together. Yesterday I finished the third module. It was a real estate prices machine learning project from start to finish. 

I had already learned every one of the libraries used in this project at Data Camp, but this program is really well structured and brings together the different tools in very practical, streamlined and results-oriented way. I feel like I am learning the way one would in an actual DS job, with a more senior practicioner sitting next to you and guiding the project from start to finish.  

#### I really recommend this program.  

An very cool feature is that the projects are done in your own machine, using the [Anaconda][anaconda] distribution of the libraries, and doing the code in [Jupyter][jupiter] notebooks. Up to now, everything I had done in Python was either in the CS50 web IDE or on the Data Camp site, where you run the code on the browser.  

I am starting to believe that little by littel I am becoming a Data Scientist. Level 0, mind you, but a Data Scientist none the less. 

#### Let's keep it going!

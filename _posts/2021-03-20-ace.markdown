I am very happy to report that I recently received my work permit. It is a one-year permit while the Green Card process is complete. I will transition at Strata (my current company) to more of an advisory role by the end of May. And I will start looking for a position in the DS field by then.

As part of that exploration, I talked to a really nice guy who graduated a couple of years ago from a data science masters at Berkley. He suggested I start learning more about the Google cloud platform. 

I really trust his advice and I signed up for the Google Certified Associate Cloud Engineer course at [A Cloud Guru][acg]. After the course one can take the certification exam. We'll see how it goes..

On the data science front, it turns out the [Elite Data Science][eds] course is just **excellent**, and it is exactly what I needed to cement what I have learned so far and to put it all together. Yesterday I finished the third module. It was a real estate prices machine learning project from start to finish. 

I had already learned every one of the libraries used in this module at Data Camp, but this program has a great structure and it brings together the different tools in very practical, streamlined and results-oriented way. I feel like I am learning the way one would in an actual DS job, with a more senior practicioner sitting next to you and guiding you through the project from start to finish.  

#### I really recommend this program.  

A very cool feature is that the projects reside in your own machine, using the [Anaconda][anaconda] distribution of the software, and runing the code in [Jupyter][jupiter] notebooks. Up to now, everything I had done in Python was either on the CS50 web IDE or on the Data Camp site, where you run the code on the browser.

I am starting to believe that little by litte I am becoming a Data Scientist. Level 0, mind you, but a Data Scientist nonetheless. 

#### Let's keep it going!
